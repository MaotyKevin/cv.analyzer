# CVMatch

An AI-powered CV analyzer that scores your CV against any job description, identifies gaps, and generates a fully rewritten, ATS-optimized version in seconds.

---

## What it does

1. Paste your CV and a job description
2. Get a match score from 0 to 100
3. See exactly what is missing and how to fix it
4. Download a fully rewritten CV tailored to that job
